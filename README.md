## Interview task for talk talk  
Code for a churn model using synthetic data

Files:
* **script.py** - main python script to run the model training and inference
* **data_loader.py** - script to import data using *read_csv()*, *read_parquet()* and *duckdb*
* **churn_dahsboard.py** - script contains function to generate single dahsboard style figure on users who churn
* **build_features.py** - script containing function to build all of the features for model
